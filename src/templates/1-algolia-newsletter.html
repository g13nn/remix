Algolia for Zendesk

A little under two years ago, we released an extension for Zendesk, our first integration with a Help Center solution. Last week, we were proud to have become an official Zendesk Guide and Zendesk Guide Enterprise partner. Here is what Algolia search brings to your Zendesk Help Center:

  • Federated search: break down content silos and empower your customers to find answers across all of your content.

  • Real-time search: direct customers to the right content with results that update with each keystroke, regardless of typos.

  • Analytics & insights: understand what drives your customers' search requests, and prioritize new Help Center content based on zero result search queries.

  • Customizability: our code is open source. Fork it, make it your own and design the experience that best fits your brand.

Read the blog post about the release, or just get started!

Release notes

React InstantSearch v5.0.0

Following our beta, the final version of React InstantSearch 5.0.0 is out. Thank you for your feedback! It includes:

  • Complete revamp of the HTML output of most widgets. The goal of this release is to provide improved semantics to our users.

  • A new CSS naming convention which will be reused across all InstantSearch libraries. This will enable the possibility to develop cross-library CSS themes easily.

  • Alignment on naming for widgets names & attributes.

You can find the full migration guide from v4 to v5 in our documentation. GitHub repo here.

InstantSearch.js 2.6.0

A lot of new features and improvements this month for our InstantSearch.js library:

  • a new widget to display results on a map (with Google Maps or any provider)

  • a widget to configure your search settings

  • support for Algolia Search Insights (click analytics)

  • the reworked pagination connector now embeds the logic for the padding of pages

Find more on our community forum.

JavaScript API client v3.25.0

To avoid confusion on pages and improve performance on old devices, our JavaScript API client will now use https by default. Follow the GitHub repo.

Vue InstantSearch 1.5.2

The last Vue InstantSearch version fixes small issues, and includes two fixes submitted by community members! Check out the changelog here, and participate in the library improvements on GitHub.

Community update

New code example: Image Box

Classifying user uploaded content is hard, especially when it’s down to the user to tell you what it is they’re sending you. In this new Algolia example, we build an image search that anyone can add to. With automatic image recognition and classification courtesy of the excellent Google Cloud Vision API.

Vue Orleans

Using the Foursquare API, our new Vue wrapper, and pun skills, we built an application to search the top visited music and drink venues in New Orleans. You can check out some of the decisions made and the code driving both. Let us know what you think!
